## 补充

tnsnames.ora 文件需要手动创建并进行相应的设置.

配置案例


```
# tnsnames.ora Network Configuration File: tnsnames.ora
# Generated by Oracle configuration tools.

ORCL_SH =
  (DESCRIPTION =
    (ADDRESS_LIST =
      (ADDRESS = (PROTOCOL = TCP)(HOST = 192.168.201.17)(PORT = 1521))
    )
    (CONNECT_DATA =
      (SERVICE_NAME = orcl)
    )
  )
```
